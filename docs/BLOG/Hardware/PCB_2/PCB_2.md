# PCB常见知识及工艺解析



## 一、相关PCB术语的概念

### 1.PCB 

印制电路板 PCB （Printed circuit boards），又称印刷电路板，是电子元器件电气连接的提供者。  
就是一块已经按电路布局要求腐蚀和铺上了阻焊层，但还没有电子元器件的铜板。

印制电路板按照线路板层数可分为单面板、双面板、四层板、六层板以及其他多层线路板。
