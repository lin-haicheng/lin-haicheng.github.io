# 程序的烧录方式



## 烧录方式

对于工程师和电子生产企业来说，给芯片写入程序，即烧录是日常工作中必不可少的工作。**离线烧录**和**在线烧录**是两种常见的数据写入方法，它们在原理、作用和使用方面有着不同的特点，因此在实际应用中需要根据具体需求进行选择。

**在线烧录适用于需要高效、低成本和自动化的生产环境，而离线烧录则更适合小批量或特殊需求（比如需烧录的程序过大，影响自动化进程）的场景。**



### **离线烧录**

**离线烧录，也称为裸片烧录（未贴到PCB板上时）**，所谓的离线烧录是先将芯片放置在各种特殊的适配座进行烧录，然后再贴片到PCB板上的生产模式。

通过适配器和不同封装的芯片链接，芯片与适配器搭配使用才能实现程序的烧录，但适配器本身也是一个精密的夹具，很难在不同的芯片之间使用，这增加了成本。一旦有不合格的产品想要重新烧录就很难将板上的芯片再拆下来，烧好后重新贴片，耗费较大的人力、物力，成本较高。在PCBA加工生产时会出现一些突发状况，如电路板耐温高度不够，拆卸芯片的时候会造成芯片变形，无形中增加报废的风险。因此，离线烧录通常不推荐用于大规模生产，所以**一般不推荐离线烧录**。



### **在线烧录**

**在线烧录，也称为在板级烧录（已经贴到PCB板上时）**，通过芯片的标准通信总线，如USB、SWD、JTAG、UART等，直接进行程序烧录（类似STM32的开发小板一样，留连接端口用ST-Link（SWD）烧录）。是对已经焊接在PCB印刷电路板上的芯片，即PCBA直接进行烧录的技术。  

它不需要烧录夹具等耗材，可通过烧录工装直接对PCB板上的芯片编程，编程后即可对板子进行功能测试。它的接口固定，连接脚位较少，通信速率适中，一般线材就可以使用。如果发现软件错误，也可以及时纠正，无需拆卸芯片，这既降低了成本，也方便经常需要软件迭代的用户。但效率相对于离线烧录来说还是没那么高。

更重要的是，生产线也在向自动化发展，越来越多的制造商将ICT、FCT等功能测试机增加在生产线上，使用自动夹具配合在线烧录的生产模式，可以在烧录阶段节省人工操作，贴板后直接烧录，然后把PCBA送到测试机进行测试，整个生产过程全流程可控、可追溯。





## 在线烧录和离线烧录怎么选择？

那么到底是应该选择在线烧录还是离线烧录？我们从以下几个方面进行分析！



### **烧录速度**

- 离线烧录通过各种适配器，对不同封装的芯片进行适配，通过最短的电路连接方式，实现对芯片的极速编程。

- 在线烧录用的是各种芯片本身的各种串行接口，如USB、SWD、JTAG、UART等，通过连接线的形式对芯片进行编程，这种方式受限于串行接口本身的性能。

不管是离线烧录还是在线烧录，不同的芯片，烧录速度也有所不同。比如目前市面上最快的烧录器AP8000，能达到50MB/s的烧录速度。



### **固定投资成本**

- 离线烧录是通过各种适配器，来兼容不同封装的芯片，而适配器本身就是属于一种精密夹具，现在越来越多的芯片朝BGA、QFN等体积小型化、封装平面化发展，这类封装测试的夹具，一般价格都不低。

- 在线烧录芯片的USB、SWD、JTAG、UART等都是标准的通信总线，接口非常固定，连接烧录的脚位也很少，并且接口通信速度不高，所以对线材的要求不高，成本低廉。



### **生产可回溯性**

- 离线烧录都是需要精密夹具来配合，如果三个部分生产测试的时候发现出错了，进行生产回溯重新修正，就需要把芯片拆卸下来，重新按照既定的流程来一遍，耗费的人力、物力较大，成本较高。

- 在线烧录就是通过连线来进行烧录，如果生产测试的时候发现出错了，立即可以对出错的PCBA进行回溯，重新再来一次即可，节省大量人力物力，成本低廉，效率高。

在正常的生产当中，还有很多额外因素，比如PCBA由于不能耐高温，在拆卸芯片的过程中很容易变形，无形中增加报废的风险。

一般来说，

对于Flash存储芯片，特别是大容量的NAND Flash，eMMC，此类芯片存储容量比较大烧录时间较长且编程时用到高速信号，此类芯片一般采用离线提前烧写，以保证整线生产效率和稳定性。

对于MCU，特别是BGA、QFN等小封装的MCU，本身体积小的特点不利于产线工人烧录，用到的精密烧录夹具价格也不菲，此类芯片适合贴板后再烧录，可减小生产成本。

正所谓“尺有所短，寸有所长”，对于工程师和生产企业来说，要根据自己的烧录需求来选择适合的烧录方式，尽量使生产效益最大化。
